Het heeft even geduurd maar nu is het zo ver. We hebben de eerste versie van ons Bash installatie script op ons Github account gezet. Zie: https://github.com/LinuXcien/AfstandsPi. Met dit script kun je een OpenVPN server maken van elke Raspberry Pi gebaseerd op de 2014-09-09 Raspbian versie (voor download zie: http://www.raspberrypi.org/downloads/. Wanneer een nieuwe Raspbian versie beschikbaar is, zullen we deze zo snel mogelijk testen. Watch this space. Lees de instructies op Github zorgvuldig. Het script schakelt aanmelden met het (voor geïnstalleerde) pi gebruiker account uit en creëert een nieuwe administratieve gebruiker met een willekeurig naam en wachtwoord. Aan het eind van de installatie rapporteert het script over de gebruiker naam, wachtwoord en IP adres geconfigureerd.
